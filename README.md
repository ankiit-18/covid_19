# COVID-19 Global Trend Analysis

## Project Overview

This repository contains a detailed analysis of global COVID-19 trends from January 22, 2020 to July 27, 2020. The project includes data preprocessing, trend analysis, mortality and recovery rate studies, WHO region comparisons, lockdown effectiveness evaluation, and visualizations created from the COVID-19 dataset.

## Repository Contents

- `COVID19_Global_Trend_Analysis.ipynb` - Main Jupyter notebook with the full analysis and visualizations.
- `COVID19_Analysis_Documentation.md` - Supporting documentation for the analysis methodology and key findings.
- `full_grouped.csv` - COVID-19 dataset used for analysis.
- `README.md` - Project overview and instructions.
- `Covid 19 Analysis A3 Document (1).pdf` - Additional project document.

## Analysis Highlights

The notebook performs the following analyses:

- Global confirmed cases, deaths, recoveries, and active case trends
- Daily new cases and deaths with moving averages
- Mortality rate and recovery rate over time
- Growth rate and doubling time analysis
- WHO region comparison for regional COVID-19 impact
- Lockdown effectiveness analysis for the US, Australia, and Germany

## How to Use

1. Open `COVID19_Global_Trend_Analysis.ipynb` in Jupyter Notebook or Jupyter Lab.
2. Ensure the dataset `full_grouped.csv` is available in the repository root.
3. Run the notebook cells sequentially to reproduce the analysis and generate plots.
